# Glossary

## Domain Terms
| Term | Definition |
|------|------------|
| SKU | Stock Keeping Unit — unique identifier for a product variant |
| OMS | Order Management System |
| RMS | Revenue Management System |
| PMS | Product Management System |
| Merchant | A business that uses MerchantOS to sell products |
| Customer | An end-user who purchases from a merchant |
| Cart | Temporary collection of items before checkout |
| Order | Completed purchase after checkout |
| Line Item | Individual product within an order |
| Fulfillment | Process of packing and shipping an order |
| Chargeback | Disputed transaction reversed by the bank |

## Technical Terms
| Term | Definition |
|------|------------|
| ADR | Architecture Decision Record — documents architectural choices and trade-offs |
| Clean Architecture | Layered architecture pattern (Domain → Application → Infrastructure → API) |
| CQRS | Command Query Responsibility Segregation — separating read and write models |
| SAST | Static Application Security Testing — analyzing source code for vulnerabilities |
| DAST | Dynamic Application Security Testing — testing running application for vulnerabilities |
| IaC | Infrastructure as Code — managing infrastructure through configuration files |
| RTO | Recovery Time Objective — maximum acceptable downtime after disaster |
| RPO | Recovery Point Objective — maximum acceptable data loss after disaster |
| C4 Model | Architecture visualization model: Context, Container, Component, Code |
| STRIDE | Threat modeling methodology: Spoofing, Tampering, Repudiation, Info Disclosure, DoS, Elevation |

## Business Metrics
| Term | Definition |
|------|------------|
| LTV | Customer Lifetime Value |
| CAC | Customer Acquisition Cost |
| ARR | Annual Recurring Revenue |
| MRR | Monthly Recurring Revenue |
| NPS | Net Promoter Score — customer satisfaction metric |
| Churn Rate | Percentage of customers who stop using the service |
| Conversion Rate | Percentage of visitors who complete a purchase |
| AOV | Average Order Value |

## Sprint/Agile Terms
| Term | Definition |
|------|------------|
| Epic | Large body of work broken into stories |
| Story | User-facing feature described from user perspective |
| Spike | Time-boxed research or exploration task |
| Velocity | Number of story points completed per sprint |
| T-shirt Size | Relative estimate: XS, S, M, L, XL, XXL |
| Definition of Done | Checklist that must be met for a story to be considered complete |
